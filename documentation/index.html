Folder Structure

  • style.css - main style.css
  • responsive.css - Adaptable in all devices
  • shortcodes.css - Html Styled Elements like buttons, lists, tabs, etc.,
  • The css folder - contains the css for all major browsers, for the font awesome. - ( Please ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours. )
  • The images folder - contains all the images that are used in html and stylesheets for the layout purpose & it has subfolder named layerslide-gallery, prettyPhoto which contains images used in blog, portfolio and in other pages
  • The js folder - contains javascript files & plugins ( Which includes)
    1. jQuery framework (jquery.js)
    2. Modernizr (modernizr-2.6.2.min.js)
    3. Isotope plugin (isotope.js)
    4. CarouFredSel plugin (jquery.carouFredSel-6.2.0-packed.js)
    5. Easing (jquery.easing.1.3.js)
    6. Tabs (jquery.tabs.min.js)
    7. bx Slider plugin (jquery.bxslider.min.js)
    8. Lightbox(jquery.prettyPhoto.js)
    9. Sticky Header Menu(jquery.sticky.js)
    10. Google Map(jquery.gmap.min.js)
    11. Twitter Feeds(jquery.tweet.min.js)
    12. Shortcodes(shortcodes.js)
    13. Form validation(validation.js)
    14. For Layer Slider
      1. jquery-transit-modified.js
      2. layerslider.kreaturamedia.jquery.js
      3. layerslider.transitions.js
      4. greensock.js
    15. And custom js like (custom.js)
  • The php folder - contains the contact form mail sending code & Mailchimp subscription form integration code.

Lists

Unordered Lists

unorderedlistclass includes tree, thumbs-up, thumbs-down, gift, tree, smile, music, moon, car, graduation-cap

Html structure for unordered lists: 

    <ul class="dt-sc-fancy-list unorderedlistclass"> 
    	<li> . . . </li>
    </ul>
Ordered Lists

orderedlistclass includes decimal, decimal-leading-zero, lower-alpha, lower-roman, upper-alpha, upper-roman

    <ol class="dt-sc-fancy-list orderedlistclass"> 
    	<li> . . . </li>
    </ol>
